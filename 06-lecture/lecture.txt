2 Адаптивне верстання

Адаптивне верстання - це верстання веб-сторінок яке забезпечує оптимальне відображення та взаємодію
сайту з користувачем незалежно від роздільної здатності та формату пристрою, з якого здійснюється 
перегляд сторінок.

У зв’язку із тим, що для відлагодження адаптивно зверстаного сайту не можливо придбати всі моделі 
мобільних пристроїв виробники браузерів забезпечують веб-розробників програмним емулятором мобільних
пристроїв, на приклад таких як Chrome Developer Tools.


3 media запити

Для створення адаптованої сторінки необхідно застосовувати певні css правила до певних елементів 
сторінки у залежності від розміру і роздільної здатності екрану. Для цього використовуються спеціальні
інструкції під назвою media запити.

Мedia запит може мати наступний вигляд:
@media (min-width: 768px) {
    ...
}

@media (min-width: 768px) означає що все правила які будуть розташовані між фігурними дужками цього 
медіа запиту будуть застосовані лише на пристроях екран яких має роздільну здатність в ширину 768рх 
або більше. Відповідно інструкції які належать до @media (max-width: 767px) будуть застосовуватись 
до пристроїв із екранами меншими за 767px або дорівнюватимуть цьому значенню. Медіа запити можуть 
застосовуватись не лише до ширини екрану, а й до його висоти. Подробиці можна дізнатись за посиланням
https://webref.ru/css/media

При створенні адаптивних сторінок варто дотримуватись правила "мобільні в першу чергу" (en "mobile 
first") яке визначає, що в першу чергу слід верстати сторінки під пристрої із найменшою роздільною 
здатністю екрану і поступово збільшуючи екран додавати відповідні css інструкції.
