1 Структура head

Документ html складається з двох основних розділів - head і body. 

У розділі head розміщуються тегі які використовуються для опису поточного документу. Крім того в 
цьому розділі відбувається підключення файлів каскадних таблиць стилів css. Також там 
можна вписати інструкції css скориставшись тегом style. Інколи у head підключають файли javascript,
але останнім часом підключати скрипти у розділі head вважається хибною практикою і пошукові машини
за це мінімізують сторінку у пошуковій видачі. Це відбувається тому, що інколи скрипт може підключатись
з якогось віддаленого сервера з яким у поточний момент відсутнє з’єднання, або воно є не стабільним.
Оскільки браузер читає html документ згори вниз рядок за рядком то якщо скрипт буде підключатись на 
початку сторінки, то цей процес може заблокувати її завантаження. Тому останнім часом скрипти підключають
у розділі body перед закриваючим тегом.
У розділі head крім всього вище перерахованого міститься тег title, через який виводиться назв сторінки

<title>Назва сторінки</title>


2 Метатегі

Які ж теги використовуються для опису документу - метаданих документу? Для цієї задачи використовується 
тег metatag з атрибутами до яких вписується відповідна інформація.

Приклади метатегів із атрибутами name та content
Кілька допустимих значень атрибуту name:
author - ім’я автора документу.
description - опис поточного документу.
keywords - ключові слова, що зустрічаються на сторінці.
viewport - управляє просмотром сайту на мобільних пристроях. Дуже важливо не забувати про нього під 
час створення адаптивної верстки.

<meta name="keywords" content="british,type face,font,fonts,highway,highways">
<meta name="description" content="Сторінка присвячена основам веб-розробки">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

Метатег що встановлює кодування сторінки:
<meta charset="utf-8"> 

Підключення файлів css
<link rel="stylesheet" href="style.css">

Підключення скрипту jQuery з віддаленого сервера
<script src="//ajax.googleapis.com/ajax/libs/jquery/3.1.0/jquery.min.js"></script>


3 Каскадні таблиці стилів css

Каскадні таблиці стилів CSS це мова, яка описує стилі елементів використаних у html документі. CSS 
описує як повинні виглядати ці елементи.

Приклад підключення таблиць стилів:
<link rel="stylesheet" href="/css/style.css">

Приклад використання таблиць стилів безпосередньо у html документі:
<style>
    body {
        background-color: green;
    }
</style>

Приклад застосування css до елементу по імені тегу:
p {
    font-size: 24px;
    color: red;
    font-weight: bold;
}

Приклад застосування css до тегів із певними атрибутами

html:
<p id="first" class="styled">Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer luctus 
ac velit at rutrum. Nam malesuada, erat ut porttitor pellentesque, elit augue lacinia turpis, et 
blandit dui dui eget quam</p>

<p class="styled">Donec a lectus laoreet, tincidunt lectus sed, pulvinar mi. Maecenas id lectus 
maximus, fringilla tortor nec, volutpat leo. </p>

<p class="styled">Donec ultrices et purus sit amet tristique. Sed maximus ultricies dui, et malesuada 
turpis blandit quis. Nulla sagittis hendrerit mi, eget maximus ipsum malesuada et. Quisque elit eros, 
tristique ut eros quis, pellentesque sagittis purus.</p>

css:

.styled {
    font-size: 26px;
}

#first {
    font-style: italic;
    color: blue;
    font-weight: bold;
}


Практика

1 Створити власний index.html

2 Заповнити head метатегами, що описують цей документ, встановлюють його сумісність із мобільними 
пристроями, і встановлюють кодування utf-8.

3 Створити заголовок першого рівня із текстом Заняття 3. Створення html документу і стилізація його 
із допомогою каскадних таблиць стилів

4 Створити заголовок другого рівня із назвою "html документ"

5 Створити параграф із текстом "html документ це текстовий документ якій розмічений із допомогою мови
гипертекстової розмітки (Hypertext Markup Language)".

6 Створити заголовок другого рівня із назвою "CSS каскадні таблиці стилів".

7 Створити параграф із текстом "Каскадні таблиці стилів CSS це мова, яка описує стилі елементів html 
документу".

8 Створити параграф із текстом "Сторінка створена на заняттях з основ веб-розробки що відбуваються  
у Політехнічному технікумі Конотопського інституту СумДУ".

9 У директорії проекту створити директорію css.

10 У директорії css створити файл style.css і підключити його у розділі head файлу index.html 

11 У файл style.css зазначити інструкцію для h1 згідно якої весь текст у цьому заголовку має бути 
заглавними літерами.

12 Для заголовків другого рівню встановити курсивний текс.

13 Для параграфів встановити розмір тексту 18 пікселів.

14 Для останнього параграфу встановити курсивний текст, колір тексту синій, задній фон цього параграфу
зробити сірим.
